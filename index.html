What is LendrClub?

LendrClub is a decentralized protocol that enables peer-to-peer lending backed by real Bitcoin reserves. Through our BTCx token, users can lend, borrow, and earn—all without intermediaries.

Key Features

  • Fractionalized Loans with Risk Mitigation
  • BTC-Backed Yield with Real Returns
  • Tiered Access Based on Token Holdings
  • DAO Governance and Transparent Operations
  • Bank-Ready API and White-Label Integrations
